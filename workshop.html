STABLE DIFFUSION INTERFACES

In order to use Stable DIffusion, you’ll need to be able to interface with it!  While many people choose to install it locally on their computer, we are recommending that all students begin their journey by using Stable Diffusion hosted on the cloud.  Cloud hosting provides professional-grade computational resources that are not available to most consumers, as well as extra features and extensions that would generally require time and labor to install and configure.  This means faster image generation, less waiting, less troubleshooting, and more learning.

Anything you learn by using cloud-hosted interfaces can be migrated to a local install if and when you decide to do your work offline.

Sources:

  • https://aituts.com/stable-diffusion-websites/

CLOUD BASED

In this section, we will get you started with using a remote computer to interface with Stable Diffusion via Automatic1111.  While there are several interfaces available, we believe that Automatic1111 is the best one to start with in order to get your bearings before moving on to other applications like ComfyUI and AiNodes.  

As we are working on the cloud, these interfaces can be used from any computer, regardless of hardware, and only require a good internet connection.

**PREFERRED CHOICE**:

Think Diffusion AI - SD 1.5, XL, ControlNet, Decorum, etc.  A1111 or ComfyUI  ~ $.50 - $2.00 / hr, Google Account Integration

Alternative Cloud-based options

Stable Diffusion WebUI (**basic text to image only - no extensions, no login required).  Good to play with before getting your feet wet.  Minimal UI with minimal functions.

Run DIffusion AI - Free and paid options, includes Automatic1111 as well as ComfyUI.  Comparable to Think Diffusion, which is our recommended app for this course.
